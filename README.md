# opencode-telegram-bridge1
This bridge enables you to use OpenCode's AI capabilities directly from Telegram. Send messages from your phone or any device, and receive intelligent responses powered by OpenCode's Sisyphus agent.这个桥接工具让你可以直接在 Telegram 中使用 OpenCode 的强大 AI 功能。随时随地发送消息，接收由 OpenCode Sisyphus agent 生成的智能回复。
